# Ru-Yi Academy

Public landing site for **Ru-Yi Academy**, a program of [Ru-Yi Meditation Center](https://www.ruyimeditation.org/).

- **Live URL:** https://academy.ruyimeditation.org
- **Hosting:** GitHub Pages
- **Phase:** Pilot (invitation-only, through 2027). Public launch planned for 2028.

## Repo layout

- `index.html` — landing page (single file, no build step)
- `CNAME` — GitHub Pages custom-domain config (`academy.ruyimeditation.org`)

Static site only. Edit HTML directly, commit, push — Pages rebuilds automatically.
